Full Stack Task & Project Management App

A full-stack MERN application for managing projects and tasks with authentication and role-based access control.

🚀 Features
JWT Authentication (Login / Signup)
Role-based access (Admin / Member)
Create & manage projects
Create, assign & track tasks
Dashboard with project/task stats
Protected routes
🛠️ Tech Stack

Frontend

React (Vite)
TypeScript
Tailwind CSS
Formik + Yup
Zustand

Backend

Node.js
Express.js
MongoDB + Mongoose
JWT Authentication

API Endpoints
POST /api/auth/signup
POST /api/auth/login
POST /api/projects
GET /api/projects
POST /api/tasks
